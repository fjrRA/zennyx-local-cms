# Zennyx Local CMS

Local content management system untuk mengelola konten website
Zennyx Interactive Studio.

## Applications

### admin-api

Backend API menggunakan:

- Express.js
- TypeScript
- Prisma
- MySQL

### admin-dashboard

Admin dashboard menggunakan:

- React
- Vite
- TypeScript
- Tailwind CSS

## Project Structure

```text
zennyx-local-cms/
├── admin-api/
└── admin-dashboard/
```
